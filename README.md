# 🌱 ESG Automation Lab

This repository contains prototypes and early development work for my ESG data automation platform. It brings together engineering domain knowledge, AI tools, and Microsoft technologies to automate ESG reporting workflows.

### 🔧 Focus Areas
- ESG data ingestion (spreadsheets, PDFs, emails)
- Document Intelligence (Azure AI)
- Data cleaning and standardization
- Scope 1/2/3 emissions modeling
- Azure Functions + automation workflows
- Power Automate orchestration
- GitHub Copilot–assisted development

This repo is where I experiment, test ideas, and build the foundation of a scalable ESG automation system.
